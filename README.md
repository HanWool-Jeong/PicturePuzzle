## 사진 퍼즐 게임
사진 조각들을 밀어서 제자리에 맞추도록 하는 게임입니다.\
걸린시간 기록, 최고기록 경신여부를 텍스트 파일에 저장, 불러와서 알려줍니다.

**따라서 오류없이 게임이 실행되도록 record.txt(최고기록을 저장하는 파일)을 꼭 프로젝트 폴더 내에 포함해 주세요**
